# 法說會簡報下載器

從公開資訊觀測站（MOPS）批次下載法人說明會簡報 PDF。

## 使用方式

雙擊 `法說會簡報下載器.bat`，依畫面指示輸入：

1. **公司代號**：例如 `2330`
2. **日期範圍**：格式 YYYYMMDD，例如 `20150101` ~ `20260325`
3. **儲存資料夾**：點「選擇」選取目標資料夾
4. 按「▶ 開始下載」

工具會自動判斷公司是上市/上櫃/興櫃/公開發行，逐月掃描並下載所有法說會簡報。

## 檔案命名

| 情況 | 檔名範例 |
|------|---------|
| 中文簡報 | `20260115_zh.pdf` |
| 英文簡報（無中文版時） | `20260115_en.pdf` |
| 同天第 2 場 | `20260115_zh_2.pdf` |

## 系統需求

- Windows 10/11
- 網際網路連線
- 首次執行時自動安裝：Python、uv、requests、beautifulsoup4

## 技術棧

Python + Tkinter / requests / beautifulsoup4 / uv
